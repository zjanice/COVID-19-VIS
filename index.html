COVID-19 Visualizations

A Database of COVID-19 Visualizations

We are creating a database of examples published online from all parts of the world.
The resulted collection of COVID-19 visualizations will be shared with anyone who want to use them for teaching and research.

How to Contribute

We are interested in ANY web page that contains a data visualization of ANY kind as long as the intention is to explain ANY aspect of the COVID-19 pandemic, in ANY language. Please add your entry via the Google form above rather than typing directly into the spreadsheet.

This project was initiated by Paul Kahn and Yixuan (Janice) Zhang @ Northeastern University
