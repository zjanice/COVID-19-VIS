Welcome to CODVIC
COVID-19 Online Data VIsualization Collection

A Database of COVID-19 Visualizations



What We Are Creating

We are creating a database of data visualizations examples related in any way to the COVID-19 Pandemic, from all parts of the world, published on the Internet. The data collection is being designed to share with anyone interested in using this for teaching and research.
We are collecting ANY web page that contains a data visualization of ANY kind meant to explain ANY aspect of the COVID-19 Pandemic.
Bar charts, line charts, maps, bubble and scatter plots, timelines, graphic simulations, infographic illustrations
ANY example of visualizing epidemic-pandemic data -- past, current or projected
ANY visualization concerning how the pandemic is affecting ANY part of the population or business sector
ANY visual explanation or instructions created in response to the pandemic: how to behave, how to prevent spreading the infection, etc.
ANY visualization of local information -- inside a county, region, department, city, institution.

How to CONTRIBUTE to the CODVIC Database

Please help us by sending us examples you find. We are interested in ANY page or post that can be accessed through a web browser containing a data visualization of ANY kind intended to explain ANY aspect of the COVID-19 pandemic, in ANY language.
The only *required* information is a valid URL. Include the optional information will speed up the coding process.
Please use this Google form to add your contribution to the collection.

How to ACCESS the CODVIC database

The collection is being managed in a spreadsheet format. We are adding examples on a daily basis. Coding the attributes of hundreds of examples is also an ongoing process. Upon request, we are happy to share a spreadsheet copy of the collection for anyone interested in using this for teaching and research.
